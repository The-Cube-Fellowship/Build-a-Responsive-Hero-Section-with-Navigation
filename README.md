## 📋 Task Assignment: Build a Responsive Hero Section with Navigation## 🎯 Objective
Create a clean, modern, and fully responsive Hero Section with a functional Navigation Bar from scratch using HTML, CSS, and vanilla JavaScript.
## 🛠️ Technical Requirements

* HTML5: Use semantic tags (<nav>, <header>, <main>, etc.) to ensure proper page structure.
* CSS3 Flexbox/Grid: Layout the navigation items and split the hero section into text content (left/top) and an image placeholder (right/bottom).
* CSS Media Queries: Ensure the layout seamlessly scales from desktop monitors down to mobile screens.
* Vanilla JavaScript: Write a script to handle the mobile menu behavior without relying on any external libraries or frameworks (like jQuery or Bootstrap).

## 📦 Feature Checklist

* Desktop Navbar: A fixed top navigation bar featuring a company logo on the left and horizontal menu links on the right.
* Mobile Navigation: At screens smaller than 768px, hide the text links and display a standard 3-line "hamburger" menu icon.
* Interactive Mobile Menu: Clicking the hamburger icon must trigger a smooth sliding or fading menu overlay containing the vertical navigation links.
* Hamburger Animation: The hamburger lines must smoothly morph into an "X" shape when the menu is active.
* Hero Content: Include a bold main heading (H1), a brief supporting paragraph, a call-to-action (CTA) button, and an eye-catching illustration or image placeholder.
* Responsive Hero Grid: The hero section should show text and image side-by-side on desktop, but stack vertically on mobile screens.

## ⏱️ Expected Deliverable
A single organized folder containing your index.html, style.css, and script.js files, or a unified index.html file containing all code sections. The code must be clean, commented, and well-indented.
